# PracticeAndYoutube

リアルVUメーター付きの練習用録音アプリです。  
練習時間を設定して録音すると、録音中の音量をVUメーターでリアルタイム表示します。  
練習終了後は指定したYouTubeチャンネルを表示できます。

---

## 機能

- 練習時間（分）を設定して録音開始
- 録音中の音量をリアルタイムで **VUメーター風アニメーション** 表示
  - バーの高さは対数スケール
  - 音量に応じて色が変化（青 → 黄 → 赤）
- 練習終了ボタンで録音停止
- 練習終了後にYouTubeキッズチャンネルを表示

---

1. Xcode でプロジェクトを開く
